A redesign of an online storefront for a small business selling art supplies and instruments. A full analysis of the current sites based on usability heuristics was conducted in order to identify core issues. The redesigned storefront has an improved colour palette and responsive design elements, addressing key issues with the previous user interface while preserving the personality of the business. A Figma prototype and full frontend implementation using HTML/CSS/Javascript were created as part of the redesign process.
